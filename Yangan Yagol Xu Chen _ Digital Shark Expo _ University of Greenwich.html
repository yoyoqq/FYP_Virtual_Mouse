Project title

Virtual Mouse

Project aim

This project aims to improve the interaction between humans and computers.

Project outline

This project shows the investigation, design, and development of a Virtual Mouse to improve the interaction between humans and computers.

The handy application enables users to control their computer using hand gestures, offering a more intuitive and natural way to interact with the device.

This project is developed as current models do not use natural gestures, making the human- computer interaction experience difficult.  The Virtual Mouse uses skeleton recognition and machine learning (ML).
